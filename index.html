Powershell Tweets

As a case study for the Windows System Administration course at Thomas More, I created a PowerShell Script that will output the tweet frequency for a specific handle (@).

It's entirely written in PowerShell and makes use of the Twitter API in order to fetch the tweets.

A user can input the handle (@) followed by the number of Tweets to fetch and above graph will be displayed.

Cryptocurrency Tracker

As a case study for the Cordova course at Thomas More, I created an Android application that allows you to monitor (track) cryptocurrencies.

The entire APP is written using HTML, CSS and JavaScript. I used the CoinGecko Cryptocurrency API to make it all work.

Some of the main features include:

  • Adding currencies to favorites
  • Searching for specific currencies
  • Sharing some cool info about that currency

Requirements Analysis

For our Requirements Analysis course we had to create an ERD an protoypes for a specifc web application.

The project was done in teams of 6 and spread out over the course of a week. During which we developed a complete ERD, and made protypes for the final application.

Our team was tasked with working out the logistics (and eventually programming) an application for people suffering from Alzheimers.

The project week eventually concluded with a jury presentation at which we had to show and explain our solution for the application.

Minesweeper Solver

I've always been really interested in challenging myself when it comes to programming, that's why I decided to write a Minesweeper Solver using Java.

The solver takes a screenshot after which it starts to figure out the positioning of the board, makes an initial random guess and then tries to solve the entire thing.

The solver was entirely written in Java and manages to solve the grids most of the time.

This project was quite challenging since I first had to figure out a way to calculate the position of the board on the screen and how big the grid is (in terms of squares).

After figuring out the board position the solver will click the center square and the game starts!
