Powershell Tweets

As a case study for the Windows System Administration course at Thomas More, I created a PowerShell Script that outputs the tweet frequency for a specific handle (@).

It's entirely written in PowerShell and makes use of the Twitter API in order to fetch the tweets.

A user can input the handle (@) followed by the number of Tweets to fetch and above graph will be displayed.

During this project I learned how to use the Twitter API, how to plot graphs using PowerShell and just how powerful of a tool PowerShell really is.
